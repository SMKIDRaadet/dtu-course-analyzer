# Privacy Policy for DTU Course Analyzer

**Last updated:** December 9, 2025

**DTU Course Analyzer** ("we", "our", or "us") is an open-source project developed and maintained by students at the Technical University of Denmark (DTU). We are committed to protecting your privacy. This Privacy Policy explains how our Chrome extension handles your data.

## 1. Data Collection
We do **not** collect, store, or transmit any personal data, browsing history, or user information to external servers.

## 2. Data Usage
The extension accesses the website `kurser.dtu.dk` solely for the purpose of:
* Reading the course number from the current URL.
* Injecting a statistics table into the webpage based on that course number.

All data analysis is performed locally on your device using a static dataset bundled within the extension.

## 3. Third-Party Services
This extension does not use any third-party analytics, tracking tools, or advertising networks.

## 4. Contact & Open Source
If you have questions about this policy, you can contact us via our GitHub repository issues page. As an open-source project, our full source code is available for review:
https://github.com/SalisMaxima/dtu-course-analyzer
